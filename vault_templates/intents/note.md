# Правила для заметок (note)

## Структура
- Заголовок h1 (`# Заголовок`) — краткий и информативный
- Основной текст — суммаризация и структурирование речи пользователя
- Если есть фото — вставляй `![[attachments/photos/filename.jpg]]` в подходящее место
- Если есть документ — вставляй ссылку `![[attachments/documents/filename]]` и опиши назначение файла
- **ВАЖНО**: Включи ВСЁ содержание записи, в том числе фрагменты на разных языках. Не пропускай текст только потому, что он на другом языке

## Пересланные сообщения
- Суммаризируй пересланное содержание (не копируй дословно)
- Добавь ссылку на источник: `> Источник: @channel_name`
- Если пользователь добавил свой комментарий — его текст обычным абзацем

## Фотографии
- Используй описание фотографии (из поля [Фото]) как контекст
- Если пользователь добавил подпись — учитывай её при описании

## Документы (файлы)
- Опиши файл по его имени и контексту от пользователя
- Вставь ссылку: `![[attachments/documents/filename.py]]`
- Если пользователь объяснил зачем отправил файл — включи это в описание
