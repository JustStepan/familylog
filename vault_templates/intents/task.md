# Правила для заданий (task)

## Структура
- Задания группируются по неделям в одном файле
- Файл: `tasks/неделя_DD-ммм-YY.md` (дата — понедельник этой недели)
- Каждое задание — строка с checkbox
- Новые задания добавляются в конец списка (append)

## Frontmatter (только при создании нового файла)
```yaml
---
tags:
  - задания
week_start: 2026-02-24
author: Степан
created: 2026-02-24 09:00
updated: 2026-02-27 11:30
---
```

## Формат заданий

### Новое задание
- Формат: `- [ ] **Описание задания** | Автор (дата)`
- Пример: `- [ ] **Купить молоко** | Степан (27.02)`

### Выполненное задание
Если пользователь использует ключевые слова: **выполнил**, **сделал**, **готово**, **закончил**, **завершил** — это означает, что задание выполнено.

- Формат: `- [x] **Описание** | Автор (дата) — выполнено`
- Пример: `- [x] **Починил кран** | Степан (27.02) — выполнено`

**ВАЖНО**: Даже если задание ранее не было записано — всё равно добавь его как выполненное с `[x]`.

## Люди в заданиях
- Если в задании упомянуты люди — добавь их в `people_mentioned`
- Новые люди (не из Памяти о семье) — в `new_people`
