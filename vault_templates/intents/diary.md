# Правила для дневника (diary)

## Структура записи
- Каждая новая запись начинается с заголовка h2: `## HH:MM — Имя`
- Пример: `## 11:30 — Степан`
- Дата берётся из поля "Дата и время" (НЕ из opened_at сессии)
- Дата в цифровом формате: "27 февраля 2026", НЕ прописью ("двадцать седьмое")

## Формат текста
- Собственный текст автора — обычные абзацы, БЕЗ цитирования `>`
- Пересланные сообщения — оформляй как цитату с `>`
- Нормализуй речь: убирай мусорные слова, исправляй грамматику
- Сохраняй личный стиль и эмоциональный тон
- **ВАЖНО**: Включи ВСЁ содержание, в том числе фрагменты на других языках

## Frontmatter
- Теги: общие для всего дня (могут обновляться при append)
- authors: список всех авторов записей за день
- created: дата создания файла (первая запись дня)

## Фотографии
- Вставляй после текста записи: `![[attachments/photos/photo_filename.jpg]]`
- Добавь краткое описание если оно уместно

## Пересланные сообщения в дневнике
- Цитируй дословно с `>`
- Указывай источник: `> Источник: @channel_name | ссылка`

## Документы
- Вставь ссылку на файл: `![[attachments/documents/filename]]`
- Краткое описание по контексту
